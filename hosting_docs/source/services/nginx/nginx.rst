Usage in CommCare
-----------------

`Nginx <https://nginx.org/en/docs/>`_ nginx [engine x] is an HTTP and reverse proxy server. It's primary use in CommCare is as a reverse proxy and load balancer however we also use it for serving static content, content caching and rate limiting.
